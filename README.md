# Kalshi Simulator (Paper Trading Dashboard)

Private personal web app prototype for paper trading live sports prediction markets.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Local Development
```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Current Scope
- Mocked seed data only.
- No external API integrations.
- Dashboard includes:
  - Live Opportunities table
  - Paper Portfolio summary
  - Open Positions table
  - Closed Trades table
  - Strategy Settings panel
  - Risk Controls panel
  - Event Log panel
